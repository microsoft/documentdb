Returns a variety of storage statistics for a given collection.

The coll_stats function provides detailed storage statistics for a specified collection.

◆ db_stats()

documentdb_api db_stats ( IN p_database_name  text,
IN p_scale float8 DEFAULT  1,
IN p_freeStorage bool DEFAULT  false 
)

Returns storage statistics for a given database.

This function returns storage statistics for the specified database, including information about collections, views, objects, and storage sizes.

Parameters
p_database_nameThe name of the database for which to retrieve statistics.
p_scaleA scaling factor for the sizes (default is 1).
p_freeStorageA boolean indicating whether to include free storage in the statistics (default is false).
Returns
BSON object containing the storage statistics for the database.

Usage Examples:

SELECT documentdb_api.db_stats('db1');
db_stats
---------------------------------------------------------------------

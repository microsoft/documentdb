Retrieves metadata and documents from a specified Mongo collection.

This function queries a MongoDB collection within the specified database, retrieving both metadata (such as shard key and object ID) and document content along with their creation times.

Usage Examples:

  • Retrieve metadata and documents from a collection:
    SELECT count(*) from documentdb_api.collection(
    'my_database',
    'my_collection');
    -- Returns:
    -- count
    -- -------
    -- 9
  • Query a non-existent collection:
    SELECT count(*) from documentdb_api.collection(
    'my_database',
    'my_collection');
    -- Returns:
    -- count
    -- -------
    -- 0
Parameters
[in]p_database_nameName of the target database. Must not be NULL.
[in]p_collection_nameName of the collection to query. Must not be NULL.
[out]shard_key_valueShard key value associated with the collection.
[out]object_idBSON object representing the object ID of the document.
[out]documentBSON object containing the document content.
[out]creation_timeTimestamp of the document's creation time.
Returns
A set of records with metadata and document information.
Note
collection function should be only used in a FROM clause

# FAQ

**Q: Is DocumentDB a fork of PostgreSQL?**  
A: No, it is a PostgreSQL extension that adds document features without modifying PostgreSQL source.

**Q: Is this the same as AWS/Azure DocumentDB?**  
A: No. This is a fully open-source PostgreSQL extension. It is not related to proprietary services with similar names.
